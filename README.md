# Shopping-Webapp

## 安装

```
# install dependencies
npm install

# Development
quasar dev

# Production(Build)
quasar build
```

## 功能与特点

- 商品列表
- 懒加载
- 轮播图
- 商品详情
- 用户评评论
- 购物车
